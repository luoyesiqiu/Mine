## 项目难点
扫雷空白块展开和生成“雷”的位置快速算法的实现
## 项目难点解决
* 1.空白块的展开使用广度优先算法实现保证展开空白块快。
* 2.生成雷的快速实现，先把地图中除点击的块外所有的位置加入到链表中，然后生成0到链表大小-1之间的随机数，根据生成的随机数在链表中取元素，取完元素就把该位置从链表中移除，并在块数组中把该块标记为。重复执行以上操作，直到生成的雷个数满足要求。
